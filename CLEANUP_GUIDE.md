# Data Cleanup Guide - GCP-Only Deployment

## ⚠️ CRITICAL: DO NOT REMOVE THESE FILES

### **Pipeline Core Files (ALL REQUIRED)**

```
pipeline/
├── gcp_main.py              ✅ REQUIRED - Flex Template entrypoint (router)
├── gcp_batch_main.py        ✅ REQUIRED - Batch pipeline (300+ lines of logic)
├── gcp_stream_main.py       ✅ REQUIRED - Streaming pipeline (320+ lines of logic)
├── schemas.py               ✅ REQUIRED - Data models (ErpInvoiceEvent, etc.)
└── transforms.py            ✅ REQUIRED - Fraud detection logic (DoFns, transforms)
```

**Why these are required:**
- `gcp_main.py` = Router (62 lines) - just delegates based on `--mode`
- `gcp_batch_main.py` = **ACTUAL batch pipeline code** (not a duplicate!)
- `gcp_stream_main.py` = **ACTUAL streaming pipeline code** (not a duplicate!)

**What happens if you delete them:**
```bash
# If you delete gcp_batch_main.py:
$ ./run_flex_template.sh
ERROR: ModuleNotFoundError: No module named 'pipeline.gcp_batch_main'

# gcp_main.py tries to import it but it doesn't exist!
```
